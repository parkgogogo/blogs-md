## English

### What is "Now I Get It"?

**Now I Get It** is an AI-powered web application that transforms dense scientific papers into accessible, interactive web pages. Created for curious people who want to understand complex research without getting lost in academic jargon.

### How It Works

1. **Upload** — Simply drag and drop or select a scientific PDF
2. **AI Processing** — The system processes the paper through multiple stages:
   - Uploading PDF
   - Security check & classification
   - Reading and analyzing the paper
   - Generating an interactive webpage
   - Publishing to the web
3. **Get Results** — Within minutes, receive a shareable, interactive web page that explains the paper in plain language

### Key Features

- **Plain Language Explanations** — Complex research translated into easy-to-understand content
- **Interactive Web Pages** — Not just text summaries, but fully interactive webpages showcasing the highlights
- **Cloud Storage** — Generated pages are stored in the cloud and accessible anytime
- **Public Gallery** — Browse a gallery of recently processed papers from the community
- **Session Persistence** — Your upload progress survives page refreshes (clears when tab closes)
- **Cost Transparency** — Shows detailed cost breakdown including input/output tokens and total cost
- **Daily Limits** — Self-funded service with a daily processing limit (resets every day)
- **Mobile-Friendly** — Responsive design with elegant dark mode UI

### Use Cases

- Students trying to understand complex research papers
- Researchers wanting to share their work with broader audiences
- Science communicators looking for accessible formats
- Curious minds exploring new fields

## 中文

### "Now I Get It" 是什么？

**Now I Get It**（意为"现在我懂了"）是一款基于 AI 的 Web 应用，它将晦涩难懂的科学论文转化为易于理解的交互式网页。专为那些想理解复杂研究又不想陷入学术术语迷宫的好奇人士打造。

### 工作原理

1. **上传** — 只需拖放或选择一篇科学 PDF 论文
2. **AI 处理** — 系统通过多个阶段处理论文：
   - 上传 PDF
   - 安全检查与分类
   - 阅读与分析论文
   - 生成交互式网页
   - 发布到网络
3. **获取结果** — 几分钟内，即可获得一个可分享的交互式网页，用通俗易懂的语言解释论文内容

### 核心功能

- **通俗语言解释** — 将复杂的研究转化为易懂的内容
- **交互式网页** — 不只是文本摘要，而是完整的交互式网页，展示论文亮点
- **云端存储** — 生成的页面存储在云端，随时可访问
- **公共画廊** — 浏览社区最近处理的论文画廊
- **会话保持** — 上传进度在页面刷新后保留（关闭标签页后清除）
- **成本透明** — 显示详细的成本明细，包括输入/输出 token 数量和总成本
- **每日限额** — 自筹资金的服务，每天有处理限额（每日重置）
- **移动友好** — 响应式设计，配备优雅的深色模式界面

### 适用场景

- 试图理解复杂论文的学生
- 希望向更广泛受众分享工作的研究人员
- 寻找易懂形式的科学传播者
- 探索新领域的好奇求知者
