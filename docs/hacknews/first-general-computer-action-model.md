# The First Fully General Computer Action Model
# 首个全通用计算机动作模型

**Source:** https://si.inc/posts/fdm1/  
**Date:** 2026-02-23

Startup Intelligence Inc. has unveiled what they claim to be the first fully general computer action model (FDM-1), capable of performing virtually any task a human can do on a computer through natural language instructions.

Startup Intelligence Inc. 推出了他们声称的首个全通用计算机动作模型（FDM-1），能够通过自然语言指令执行人类在计算机上能做的几乎任何任务。

Unlike previous systems that were limited to specific applications or predefined workflows, FDM-1 operates at the OS level, interacting with any software through the same interfaces a human would use. This generalization represents a significant leap toward truly autonomous computer agents.

与之前仅限于特定应用或预定义工作流的系统不同，FDM-1 在操作系统级别运行，通过人类使用的相同界面与任何软件交互。这种泛化代表着向真正自主计算机代理的重大飞跃。

Key capabilities demonstrated:
- Cross-application workflows spanning multiple programs
- Adaptation to new software without specific training
- Understanding of GUI elements and their semantic meaning
- Natural language instruction following with context awareness

展示的主要能力：
- 跨多个程序的跨应用工作流
- 无需特定训练即可适应新软件
- 理解 GUI 元素及其语义含义
- 具有上下文意识的遵循自然语言指令

The model uses a novel architecture combining computer vision, natural language understanding, and reinforcement learning to navigate complex software environments. It can learn from demonstrations and generalize to new tasks within similar domains.

该模型使用一种新颖的架构，结合计算机视觉、自然语言理解和强化学习来导航复杂的软件环境。它可以从演示中学习，并在相似领域内泛化到新任务。

Safety considerations have been central to FDM-1's development, with built-in constraints preventing unauthorized access to sensitive systems and requiring explicit user confirmation for irreversible actions.

安全性考虑一直是 FDM-1 开发的核心，内置约束防止未经授权访问敏感系统，并要求对不可逆操作进行明确的用户确认。

While still in research preview, FDM-1 demonstrates capabilities that could transform how humans interact with computers, potentially eliminating the need for specialized training on complex software tools.

虽然仍处于研究预览阶段，FDM-1 展示的能力可能会改变人类与计算机交互的方式，潜在地消除了对复杂软件工具进行专门培训的需要。
