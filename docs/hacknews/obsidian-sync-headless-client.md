# Obsidian Sync 现已支持无头客户端

**Original:** https://help.obsidian.md/sync/headless  
**Rating:** 8/10 | **Category:** Dev

---

## English Content

Obsidian released an official headless CLI client for Obsidian Sync, enabling users to synchronize vaults from command line without the desktop app.

Key features:
- Official headless CLI client
- Command-line vault synchronization
- No desktop app required
- Enables automation workflows
- CI/CD integration support
- Server-side syncing for developers

This is a significant development for developers who want to:
- Automate backup workflows
- Integrate with CI/CD pipelines
- Run sync operations on servers
- Build custom automation around Obsidian

主要功能：
- 官方无头 CLI 客户端
- 命令行知识库同步
- 无需桌面应用
- 支持自动化工作流
- CI/CD 集成支持
- 开发者服务器端同步

对于希望以下操作的开发者来说，这是一个重大进展：
- 自动化备份工作流
- 与 CI/CD 管道集成
- 在服务器上运行同步操作
- 围绕 Obsidian 构建自定义自动化
